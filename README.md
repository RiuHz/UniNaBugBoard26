# 🇬🇧 UniNaBugBoard26

UniNaBugBoard26 is a bug management system developed to track, manage, and resolve issues within a software project. This project was created as part of a university assignment for simple and structured bug management and monitoring.

The system allows users to:

- Report bugs with a description and a priority.
- View reported bugs in a list.
- Assign a status to the bugs to track their progress (e.g., "To Do", "In Progress", "Resolved").
- Modify or delete reported bugs if necessary.

*This repository contains the work done for the Software Engineering exam.*

# 🇮🇹 UniNaBugBoard26

UniNaBugBoard26 è un sistema di gestione dei bug sviluppato per tracciare, gestire e risolvere i problemi all'interno di un progetto software. Questo progetto è stato realizzato come parte di un'attività universitaria per la gestione e il monitoraggio dei bug in modo semplice e strutturato.

Il sistema permette agli utenti di:
- Segnalare bug con una descrizione e una priorità.
- Visualizzare i bug segnalati in una lista.
- Assegnare uno stato ai bug per monitorare il loro avanzamento (ad esempio: "Da fare", "In corso", "Risolto").
- Modificare o eliminare i bug segnalati, se necessario.

*Questo repository contiene il lavoro svolto per l'esame di Ingegneria del Software.*
